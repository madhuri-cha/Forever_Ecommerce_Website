## 🚀 Features

### 👤 User Experience
✨ Browse products with ease  
🔍 Search & filter functionality  
🛍️ Add to cart & manage items  
📱 Fully responsive design  

### 🔐 Authentication
🔑 Secure login & registration  
🪪 JWT-based authentication  

### 🛒 Cart System
➕ Add / Remove products  
🔄 Update quantity dynamically  
💰 Real-time price calculation  

### 📦 Orders
📥 Place orders  
📄 View order summary  
✅ Order confirmation  

### 🛠️ Admin Panel
📊 Manage products  
✏️ Edit & delete items  
📦 Track all orders
